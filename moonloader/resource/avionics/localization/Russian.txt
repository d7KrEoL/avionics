Список текстовых команд скрипта
Данные команды вводятся в общий чат, как обычные команды сервера:
/setppm /setwpt - установить текущий ППМ
/swavionics /swav /avionix - Открыть/закрыть данное меню
/swcam - Перейти в контейнер целеуказания/выйти из него
/swmag - (Для вертолётов) выпустить/сбросить магнит
/addwpt /addppm - Добавить ППМ по координатам
/clearppm /clearwpt - Очистить план полёта (удалить все ППМ)
/autopilot /swapt - Включить автопилот
/swapto - Отключить автопилот
/wptcam /ppmcam /tarcam - Повернуть камеру контейнера целеуказания на ППМ
/tarppm /tarwpt - Добавить ППМ из координат цели \n(фиксированной камеры)
/vehppm /vehwpt - Добавить ППМ из текущих координат самолёта \n(собственных)
/swamode /swam - Переключение режима авионики
/swazoom /swaz - Изменить скорость приближения камеры
/safp /ldfp - Загрузить план полёта из файла (ложить в папку\nresource/avionics/flightplan)
/savefp /svfp - Сохранить план полёта в файл (будет лежать в папке\nresource/avionics/flightplan)
